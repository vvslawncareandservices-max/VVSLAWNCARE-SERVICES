# VVS Lawncare & Services - Official Website

A colorful, modern, and mobile-friendly lawn care website built with pure HTML, CSS, and JavaScript. Designed to be lightweight, ad-free, and professional, reflecting the company's brand and values while providing customers with fast, easy access to services.

## Features

✅ **Instant Quote Request Form** - Customers can quickly request quotes with detailed service information  
✅ **Customer Referral Program** - Built-in referral form to help grow business through word-of-mouth  
✅ **"Call Now" Button** - Prominent call-to-action for immediate phone contact  
✅ **Email Contact Form** - Professional contact form for general inquiries  
✅ **Mobile-Friendly Design** - Fully responsive layout that works on all devices  
✅ **Modern UI/UX** - Clean, professional design with branded green color scheme  
✅ **FormSubmit Integration** - All forms send emails directly to the business owner  
✅ **GitHub Pages Ready** - Optimized for free hosting on GitHub Pages  

## Services Highlighted

- Lawn Mowing
- Landscaping
- Tree Care
- Irrigation Systems

## Technical Stack

- **HTML5** - Semantic, accessible markup
- **CSS3** - Modern styling with CSS Grid and Flexbox
- **JavaScript** - Enhanced user experience and form functionality
- **FormSubmit** - Free form processing service
- **Font Awesome** - Professional icons
- **Google Fonts** - Clean, modern typography

## Getting Started

1. Clone this repository
2. Update the FormSubmit email addresses in the HTML forms (replace `your-email@example.com`)
3. Customize contact information (phone numbers, email addresses)
4. Deploy to GitHub Pages or any static hosting service

## Customization

- Update contact information in `index.html`
- Modify colors in the CSS `:root` variables
- Add or remove services in the services section
- Customize form fields as needed
